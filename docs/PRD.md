## 1. 제품 비전

제조업체가 사내망에서 안전하게 도면을 **조회, 측정, 마크업**할 수 있는 웹 기반 CAD 뷰어.

별도의 CAD 소프트웨어 라이센스 없이, 브라우저만으로 DWG/DXF 도면을 열고 협업할 수 있다.

---

---

## 3. 보안 요구사항 (P0 — 최우선)

1. **파일 비전송 원칙**: DWG/DXF 파일은 절대 외부 서버/클라우드로 전송하지 않는다
2. **클라이언트 사이드 처리**: 모든 도면 파싱 및 렌더링은 브라우저(클라이언트)에서 수행한다
3. **온프레미스 배포**: 사내망 전용 배포를 기본으로 설계한다
4. **번들 자급자족**: 서드파티 CDN 의존성 없이 모든 라이브러리를 번들에 포함한다
5. **오프라인 동작**: 네트워크 연결 없이도 핵심 기능(파일 열기, 뷰잉)이 동작한다
6. **CSP 적용**: Content-Security-Policy 헤더로 외부 도메인 연결을 차단한다

## 7. UI 구조 (Figma 프로토타입 기반)

```
┌─────────────────────────────────────────────────────┐
│                    Ribbon (상단)                      │
│  [File ▼] [Draw] [Modify] [Annotation] [Layers]     │
├────┬────────────────────────────────────┬────────────┤
│Left│                                    │ Properties │
│Tool│         ViewerCanvas               │   Panel    │
│bar │           (중앙)                    │   (우측)   │
│    │                                    │            │
├────┴────────────────────────────────────┴────────────┤
│              CommandPalette (명령줄)                   │
├─────────────────────────────────────────────────────┤
│              StatusBar (좌표, 스냅, 줌)                │
└─────────────────────────────────────────────────────┘
```

**핵심 컴포넌트**:
- **Ribbon**: 상단 메뉴바 + 리본 패널 (Draw/Modify/Annotation/Layers)
- **LeftToolbar**: 좌측 수직 도구 팔레트
- **ViewerCanvas**: 중앙 CAD 뷰포트 (@mlightcad/cad-viewer 마운트)
- **PropertiesPanel**: 우측 속성 패널 (선택 객체 정보)
- **CommandPalette**: 하단 명령줄 (AutoCAD 스타일)
- **StatusBar**: 하단 상태바 (좌표, 스냅 토글, 줌 레벨)

## 10. 코딩 컨벤션

- TypeScript strict mode 사용
- 컴포넌트 파일명: PascalCase (예: `CadViewer.vue`)
- 함수/변수: camelCase
- 상수: UPPER_SNAKE_CASE
- 테스트 파일: `*.test.ts` (Vitest)
- 커밋 메시지: Conventional Commits (`feat:`, `fix:`, `docs:`, `style:`, `refactor:`)
- CSS: Tailwind 유틸리티 클래스 우선, 커스텀 CSS는 최소화

## 11. 제약 및 가정

### 제약
- @mlightcad/cad-viewer의 DWG 버전 지원 범위에 의존
- 브라우저 메모리 제한 (대용량 DWG 파일 처리 시)
- 온프레미스 전용 — SaaS 배포 불가

### 가정
- 사용자는 Chrome 90+ 또는 Edge 90+를 사용한다
- DWG/DXF 파일은 사용자의 로컬 PC 또는 사내 네트워크 드라이브에 존재한다
- @mlightcad/cad-viewer 라이센스가 상용 사용에 적합하다
