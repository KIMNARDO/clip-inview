# M1 스프린트 계획서 — DWG 파일 열기 + 기본 뷰잉

| 항목 | 내용 |
|------|------|
| 스프린트 | M1 (Sprint 1) |
| 기간 | 2주 (10 영업일) |
| 목표 | 사용자가 로컬 DWG/DXF 파일을 브라우저에서 열고, Pan/Zoom/Fit으로 조회할 수 있다 |
| 핵심 제약 | 파일은 FileReader API로 로컬 로드, 외부 전송 절대 불가 |

---

## 1. 스프린트 목표

1. 프로젝트 기초 환경 구축 (pnpm + Vite + Vue 3 + TypeScript + Tailwind CSS)
2. @mlightcad/cad-viewer 라이브러리 통합 및 DWG/DXF 렌더링
3. Figma 프로토타입 기반 핵심 UI 셸 구현 (Ribbon, Canvas, StatusBar)
4. 기본 뷰포트 인터랙션: Pan, Zoom, Fit-to-Extents
5. 파일 열기 플로우: 로컬 파일 선택 → 파싱 → 렌더링

---

## 2. 사용자 스토리

### US-01: 파일 열기
> **As a** 제조현장 엔지니어,
> **I want** 브라우저에서 내 PC의 DWG/DXF 파일을 선택하여 열 수 있기를,
> **So that** 별도 CAD 소프트웨어 설치 없이 도면을 조회할 수 있다.

### US-02: 뷰포트 탐색 (Pan/Zoom)
> **As a** 도면 조회 사용자,
> **I want** 마우스 드래그로 Pan, 휠로 Zoom을 할 수 있기를,
> **So that** 도면의 원하는 영역을 자유롭게 탐색할 수 있다.

### US-03: 전체 보기 (Fit-to-Extents)
> **As a** 도면 조회 사용자,
> **I want** 버튼 클릭 한 번으로 도면 전체가 화면에 맞춰지기를,
> **So that** 도면 전체 형상을 한눈에 파악할 수 있다.

### US-04: 좌표/줌 상태 확인
> **As a** 엔지니어,
> **I want** 하단 상태바에서 현재 마우스 좌표와 줌 레벨을 실시간으로 확인하기를,
> **So that** 도면 내 위치를 정확히 파악할 수 있다.

### US-05: 보안 확인
> **As a** 관리자,
> **I want** 도면 파일이 외부 서버로 전송되지 않음을 보장받기를,
> **So that** 사내 기밀 도면의 유출 위험이 없다.

## 4. 수용 기준 (Acceptance Criteria)

### AC-01: 파일 열기 (US-01)
- [ ] "파일 열기" 버튼 클릭 시 OS 파일 선택 다이얼로그가 열린다
- [ ] .dwg, .dxf 확장자 파일을 선택하면 캔버스에 도면이 렌더링된다
- [ ] 파일을 캔버스 영역에 드래그 앤 드롭으로도 열 수 있다
- [ ] 지원하지 않는 파일 형식 선택 시 에러 메시지가 표시된다
- [ ] 대용량 파일(50MB+) 로드 시 프로그레스 인디케이터가 표시된다
- [ ] **[보안]** 브라우저 DevTools Network 탭에서 파일 관련 외부 요청이 0건이다

### AC-02: Pan/Zoom (US-02)
- [ ] 마우스 휠 스크롤로 줌 인/아웃이 동작한다
- [ ] 줌은 마우스 커서 위치를 중심으로 수행된다
- [ ] 마우스 중간 버튼 드래그로 Pan이 동작한다
- [ ] Pan/Zoom 시 렌더링 프레임 드롭 없이 부드럽게 동작한다 (30fps 이상)

### AC-03: Fit-to-Extents (US-03)
- [ ] Ribbon 또는 단축키(Home)로 Fit-to-Extents를 실행할 수 있다
- [ ] 실행 시 도면 전체가 캔버스 영역에 여백과 함께 맞춰진다
- [ ] 파일 최초 로드 시 자동으로 Fit-to-Extents가 적용된다

### AC-04: 상태바 (US-04)
- [ ] 마우스 이동 시 하단 상태바에 CAD 좌표(X, Y)가 실시간 갱신된다
- [ ] 현재 줌 레벨(%)이 상태바에 표시된다

### AC-05: 보안 (US-05)
- [ ] FileReader API만으로 파일을 로드하며, XMLHttpRequest/fetch로 파일을 전송하지 않는다
- [ ] Content-Security-Policy 헤더가 설정되어 외부 도메인 연결이 차단된다
- [ ] 서드파티 CDN에서 로드되는 리소스가 없다 (모든 의존성 번들 포함)
